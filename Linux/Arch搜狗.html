安装fcitx

fcitx是输入法的框架，是必须安装的。安装方法：

sudo pacman -S fcitx fcitx-im fcitx-configtool

fcitx：这个是输入法的框架，fcitx-im：这个是fcitx的模块必须全部安装，fcitx-configtool：fcitx的图形化配置工具。

安装fcitx-qt4

默认fcitx-qt4这个模块是不包括在fcitx-im软件包组里面的，需要手动安装。其他输入法没有fcitx-qt4是可以用的，但搜狗不行。因此必须安装。

手动从AUR 安装

git clone https://aur.archlinux.org/fcitx-qt4.git ~fc
cd fc
makepkg -si

用yay安装

-S fcitx-qt4

安装搜狗输入法

接下来安装搜狗输入法本体

yay -S fcitx-sogoupinyin

AUR上搜狗无法安装的解决方法

添加ArchCN源

/etc/pacman.conf

在里面加入

[archlinuxcn]
SigLevel = Never
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

接下来导入密钥

pacman -S archlinuxcn-keyring

更新软件源并安装搜狗输入法

pacman -Sy fcitx-sogoupinyin

最后一步，添加环境变量

如果你是使用显示管理器的用户，在~.pam_environment里添加这三行：
GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
如果你是使用startx手动启动桌面环境，那么请将这三行加入~.xinitrc

至此，完成
